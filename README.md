# CRUD-Application
This project is a simple implementation of User Registration and Login Module using Spring Boot and Thymeleaf.

# Requirements
- Java 17
- Spring Boot 3
- Spring MVC 6
- Spring Security 6
- Hibernate 6
- Thymeleaf 3
- MySQL 8
- Maven
